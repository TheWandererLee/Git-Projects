Instructions on Using Files

  1. Click the name of the file to download it.
  2. Save the file to the directory 'C:\Program Files\Datel\Action Replay XBOX\Saves' (if you have Action Replay)
  3. Open Action Replay
  4. Find the folder 'Halo 2' in the 'PC Database' section of Action Replay
  5. Open the folder, and find the game that you downloaded
  6. Drag the file from the PC Database onto the 'Memory Card' section
  7. If it won't let you drag it, right-click the file, then left-click the file and drag it, repeat until it works
  8. Plug the memory card into your Xbox Controller, turn on the Xbox with no game inserted
  9. Select 'Memory', choose the memory card, find the file and choose copy to copy it to your hard drive
  10. Play Halo 2, and choose the Game Variant, (it might not be named the same as it is here)
  11. Have Fun!
